02 / 12
📖

기존 방식의 어려움

  • 업무일지 · 근태 따로따로 (엑셀 · 카톡 · 메모)
  • 법정점검, 공문서 마감일 놓치기 쉬움
  • 프로젝트 진행상황 한눈에 안 보임
  • 직원 정보 · 부서 관리 산발적
  • 부서장 보고용 자료를 매번 새로 정리
✅

이 시스템 도입 후

  • 업무 · 근태 · 프로젝트 · 점검 · 공문서 한 화면
  • 마감일 자동 알림 (D-Day 표시)
  • 업무일지 자동 보고서 생성 (당일 / 주간 / 월간)
  • Google 계정으로 안전하게 로그인
  • 모바일에서도 동일하게 사용 가능
한마디로: 흩어져 있던 회사의 일을 한 곳에 모아서, 빠뜨리지 않고 관리할 수 있게 해주는 도구입니다.

업무일지 — 매일 아침 5분

05 / 12

🕒 출근 후 바로 등록

  • 오늘 할 일을 3종 유형으로 분류
  • 정기업무 · 프로젝트 · 돌발업무
  • 제목 + 설명만 간단히 입력
  • 업무 끝나면 동그라미 클릭으로 완료 처리

⚡ 시간을 줄이는 기능

  • "어제 복사" — 반복 업무 클릭 한번에
  • 업무 등록 후 제목/설명 편집 가능
  • 휴일 · 주말 자동 처리 (특근 별도 등록)
  • 모바일에서도 동일하게
마감 시간: 매일 오전 8:30까지 작성 → 자동 알림 발송 (이메일).
연차 · 병가 · 반차(오전) 사용자는 자동 예외 처리됩니다.

안전한 로그인 · 안전한 데이터

11 / 12

🔒 로그인

  • Google 계정으로 로그인 — 별도 비밀번호 없음
  • 회사가 사전 등록한 직원만 접근 가능 (화이트리스트)
  • 외부인이 로그인 시도 시 자동 차단
  • 비활성 계정은 즉시 거부

🔐 데이터 보호

  • 역할 기반 권한 (admin · manager · staff)
  • 모든 통신 HTTPS 암호화
  • 샘플 모드 — 실데이터 없이 체험 가능
  • 승인 워크플로우 — 누가 언제 승인했는지 기록
장점: Google 계정 비밀번호는 Google이 관리 — 회사가 비밀번호를 보관하지 않습니다.
직원이 Google에서 2단계 인증을 사용하면 그 보안 그대로 적용됩니다.
